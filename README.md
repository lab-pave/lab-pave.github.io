# Website Test Suite

- A tiny static site scaffold to verify GitHub Pages hosting, routing, and asset delivery.

How to test locally:
- Run a simple static server, for example:
  - python -m http.server 8000
  - or npx http-server -p 8000
- Open http://localhost:8000/index.html

How to publish:
- Push this directory to a GitHub Pages repository following the plan described earlier.
